Those are ~all my dotfiles, install them if you want via GNU Stow:
```
git clone https://github.com/luclu7/dotfiles ~/dotfiles
cd ~/dotfiles
stow -v vim
stow -v i3
stow -v termite
stow -v bins
stow -v fish
```
